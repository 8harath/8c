## How to Add Topics to GitHub

1. Navigate to your repository: `https://github.com/8harath/8c`
2. Click the **⚙️ gear icon** next to "About" (top right of repository page)
3. In the "Topics" field, add topics one by one (or paste comma-separated)
4. Click **"Save changes"**

**Note**: Topics must be:
- Lowercase
- Alphanumeric (hyphens allowed, no spaces)
- Max 50 characters each
- Max 20 topics total
